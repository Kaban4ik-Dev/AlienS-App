# 👽 AlienS App
AlienS App is a cross-platform application designed to facilitate introductory and intermediate-level programming education. 
The app provides structured coding lessons, interactive exercises, and progress tracking in a unified interface. 
Its goal is to provide an accessible, hands-on learning experience for individuals aiming to develop foundational software development skills.

# File structure after the build
AlienSApp
- assets # stores all images, fonts and sounds
- bin
  - aliens_app.exe
  - ... and all other build results
- courses
  - download  # courses from website
  - example   # course examples
  - local     # user-created corses
- data
  - code # folder with code execution applications
    - temp_files # folder where input and output files are temporarily placed
    - c_cpp
    - python
    - verilog
  - config # stores configuration files
  - user # stores account data
